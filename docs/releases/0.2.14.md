# AT Terminal 0.2.14 Release Notes

## Summary

Version 0.2.14 focuses on SSH host-key trust recovery and the server group combobox in the add/edit server form.

## Fixed

- Clears persisted trusted host key when a server is deleted, so recreating a server after an OS reinstall can prompt for the new fingerprint instead of staying blocked by the old trust record.
- Replaces the native group datalist with an anchored combobox to keep the dropdown aligned with the Group field inside the VS Code webview.
- Keeps typed group input filtering while manual dropdown expansion shows all groups.
- Prevents manual group dropdown expansion from being immediately re-filtered when focus returns to the input.
